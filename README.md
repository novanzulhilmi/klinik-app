# <p align="center">Tugas Klinik App</p>

## Fitur yang tersedia :

- Tampilan Depan (welcome)
- Halaman Register
- Halaman Login
- Dashboard (home)
- Lihat Data Pasien
- Tambah data dengan foto
- Tambah data tanpa foto
- Edit Data
- Delete Data

## Fitur yang diupdate :
- Edit Data
- Delete Data
- Plus     : Gambar berhasil terupdate
- Minus    : Gambar sebelumnya tidak terhapus, masih berada di folder "Storage"
